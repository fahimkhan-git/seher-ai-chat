Copy and paste this script into your live microsite to test the AI-powered chat widget

üìã How to Use:

  • Copy the script below
  • Paste it just before the closing </body> tag on your microsite
  • Replace YOUR_PROJECT_ID with your actual project ID (e.g., 5796, lodha-park, etc.)
  • Replace YOUR_MICROSITE_NAME with your microsite identifier
  • The chat will auto-open after a few seconds

Option 1: Production (Vercel Deployed)

<!-- Homesfy Chat Widget with AI -->
<script
  src="https://widget-eight-ebon-chi.vercel.app/widget.js"
  data-project="YOUR_PROJECT_ID"
  data-api-base-url="https://api-alpha-ten-43.vercel.app"
  data-microsite="YOUR_MICROSITE_NAME"
  async
  onload="window.HomesfyChat && window.HomesfyChat.init && window.HomesfyChat.init({ element: this });"
></script>
